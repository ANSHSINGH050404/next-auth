# next-auth

NextAuth authentication examples and auth flows for Next.js.

## Stack

Next.js, TypeScript

## Getting started

```bash
git clone https://github.com/ANSHSINGH050404/next-auth.git
cd next-auth
npm install
npm run dev   # or: npm start
```
